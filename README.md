# Build Up
아마추어도 **풋살, 축구를 **제대로 즐기고** 싶으니까

## Stack
- React & Vite
- Typescript
- Tanstack-Query(react-query)
- Tailwind-css
- Zustand

<br/>

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
